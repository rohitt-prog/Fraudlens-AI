# Dataset Specifications

## Data Directory Layout
Data resides inside `ml/data/` (Gitignored):
- `raw/`: Uncleaned, primary transaction data.
- `processed/`: Fitted arrays and engineered target variables.
- `external/`: Auxiliary records such as geographical maps or blacklist IP database indexes.
