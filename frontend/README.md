# FraudLens AI - Frontend Dashboard

This directory is reserved for the Next.js and React frontend dashboard for FraudLens AI, which will be generated using Stitch MCP.

## Directory Structure

```
frontend/
├── app/                 # Next.js App Router pages and layouts
├── components/          # Reusable React components (buttons, cards, graphs)
├── lib/                 # Common frontend utility functions and API clients
├── public/              # Static assets (images, icons, vectors)
├── styles/              # Global styles and tailwind configs
└── package.json         # Javascript dependencies and script runner configurations
```

## Setup and Run

To set up and run the frontend application locally:

```bash
cd frontend

# Install dependencies
npm install

# Run the dev server
npm run dev
```

The application will be accessible at [http://localhost:3000](http://localhost:3000).
